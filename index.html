Download Instructions

Step 1: Download the alphaconsole plugin for bakkesmod. Click to go to download page
Step 2: Click "Open plugininstaller.exe"
Step 3: Once the mod has downloaded, make sure the alphaconsole plugin is activated by checking the box.
Step 4: Download the decal zip file you want above
Step 5: Open bakkesmod and go to "File" and click "Open Bakkesmod Folder"
Step 6: In the bakkesmod folder, go to "data > acplugin > DecalTextures" and paste the zip file there
Step 7: Extract the file in "DecalTextures". The target file directory should look something like this
Step 8: To apply the decal, go in game and pick a car (the car the decal is made for) with no decal on it and press F5 to open the alphaconsole and simple choose the decal you want
 
IF YOU WANT TO USE DECALS WITH GLOSS YOU NEED THE MOIST ESPORTS 2022 HOME FENNEC (skinID: 8077)
If you don't have the Moist Esports Fennec, you NEED an ESPORTS DECAL with AT LEAST some anodized or glossy parts on it
 
Step 9: Open the bakkesmod console by pressing F6, and type in "acplugin_texturedeveloper_enabled 1"
Step 10: Equip an esports decal that has some glossy or anodized parts and open alphaconsole again by pressing F5
Step 11: Find the Decal ID in the alphaconsole for the decal you have equipped. Here I have a Karmine Corp Fennec with Decal ID 8371.
Step 12: Go back to the folder containing the decals (from Step 7) and open the file named "_TexturesPackage.json" and right click on the file.
Step 13: Click "Open with" and choose "Notepad". If you can't see the notepad option, click choose another app and the option should appear.
Step 14: Change the skinID value (originally 8077) to the Decal ID of the esports decal you have equipped. DO NOT remove any commas as it will break the json syntax and it will not work
Step 15: Restart the game and make sure that you have your esports decal (that has the same Decal ID as the number you have entered into the json file) and simply equip a glossy version of the decal in the alphaconsole.
Custom decals with no gloss - Only works with cars with no decal on it
Glossy custom decals - Only works with cars with a Decal ID that match the skinID in the json file
